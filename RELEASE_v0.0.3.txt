📋 RELEASE NOTES v0.0.3

🔧 AJUSTES E CORREÇÕES:

✅ Ajustes de pop-ups para se ajustar às telas

═══════════════════════════════════════════════════════════

📦 COMO ATUALIZAR:

1. Edite seu arquivo docker-stack.yml

2. Atualize as tags das imagens:

   Frontend:
   image: astraonline/astracampaignfrontend:0.0.4

   Backend:
   image: astraonline/astracampaignbackend:0.0.3

3. Execute o comando de atualização:
   docker stack deploy -c docker-stack.yml astracampaign

4. Aguarde alguns instantes para o deploy ser concluído!
